Learning Management System Project

Add Task:

  • In Add Task you can create your individual tasks.
  • All fields marked with * must be filled in.
  • Please fill out the following fields:
    • Title
    • Date
    • Category
    • Urgency (High, Medium, Low)
    • Description
    • Assign to (choose a User)

Backlog:

  • In the backlog, you can view the created tasks.
  • The color marking and symbol of the tasks symbolizes the importance of the task according to the traffic light principle.
  • Once a task has been selected, the possibility exists:
    • Use "Add to Board" to move the task to the board

Board

  • On the board you have the possibility to sort all your created tasks.
  • The individual small boards are available to them for this purpose.
  • Tasks can be moved between the boards via "drag and drop".
  • As in the backlog, the color coding shows the importance of the task according to the traffic light principle.
